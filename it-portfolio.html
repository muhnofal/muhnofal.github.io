Sunnah Life

Sunnah Life adalah aplikasi komunitas islami yang menjunjung Sunnah Rasulullah. Ini adalah project pertama saya ketika saya pertama kali bekerja fulltime. Di project ini saya belajar banyak mengenai Flutter, seperti Getx, slicing UI yang bagus, clean code, API Consume dan masih banyak lagi. Disini juga saya belajar banyak mengenai kolaborasi antara sesama programmer ataupun dari tim lain seperti UI/UX.

Flutter, Firebase, API, Getx, Google Maps, UI/UX

Cook Kuy

Cook Kuy adalah aplikasi yang bertujuan mencari resep makanan. User hanya perlu mencari resep makanan dengan menginput bahan yang dipiih dan aplikasi akan mencari sesuai dengan inputan user. Ini adalah project skripsi saya dan teman saya. Disini saya menggunakan Flutter sebagai framework. Tugas saya dalam pembuatan project ini hampir semua. Integrasi Firebase, login register, slicing UI/UX, fitur search dan masih banyak lagi.

Flutter, Firebase

FIXYOU

PORTO HRIS adalah project yang saya kerjakan ketika magang di perusahaan Porto. Disini saya pertama kali untuk menggunakan Flutter untuk membangun aplikasi. Disni saya belajar mengenai state management BLOC, Git dan consume API.

Flutter, BLOC, GIT, API

FIXYOU

FIXYOU adalah project yang saya dan tim kerjakan di program Bangkit. Aplikasi ini bertujuan mencari psikolog secara online. Di project ini kami terbagi menjadi 3 role dan salah satunya adalah Mobile Developer. Disini saya dan teman saya menggunakan Kotlin dan berkolaboraasi menggunakan GIT, dan untuk databasenya kami menggunakan Firebase. Kami membagi tugas kami per module agar tidak saling bertabrakan saat mendevelop fitur - fitur pada aplikasi ini.

Android studio, Kotlin

Let's Edit

Let's Edit adalah aplikasi mobile pertama saya yang saya kerjakan. Dengan menggunakan Android studio dan bahasa Java, saya dan beberapa tim saya bisa membuat aplikasi ini. Tugas saya dalam pembuatan aplikasi ini yaitu semua, mulai dari koneksi ke Firebase, mengambil data dari Firebase yang dibuat dan tentunya slicing UI ke aplikasi. Cukup bangga dengan project pertama ini, karena bisa membuat aplikasi mobile pertama saya, ya walaupun dari segi kodingan masih acak-acakan.

Android studio, Java
